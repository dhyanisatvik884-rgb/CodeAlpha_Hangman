# CodeAlpha Python Programming Tasks

This repository contains the Python projects completed as part of the **CodeAlpha Python Programming Internship**. Each project demonstrates different programming concepts including game development, automation, data handling, and chatbot creation.

## 📌 Projects Included

A console-based Hangman game where players guess a hidden word letter by letter.

#### Features
- Random word selection
- Limited attempts system
- Input validation
- Win/Loss detection

#### Skills Used
- Python Loops
- Conditional Statements
- String Manipulation
- Lists
- Random Module
